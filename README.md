# Sm0lJ-stromy-counter

Statická webová aplikácia publikovaná cez GitHub Pages.

## Publikovanie

Repo obsahuje workflow `.github/workflows/pages.yml`, ktorý pri pushi na `main` nasadí obsah repozitára na GitHub Pages.

## Použitie

1. Klikni na Otvoriť kameru.
2. Použi Odfotiť a spočítať (alebo samostatne Odfotiť a potom Spočítať stromčeky).
3. Ak výsledok nesedí, dolaď Citlivosť, Min. polomer a Max. polomer.
